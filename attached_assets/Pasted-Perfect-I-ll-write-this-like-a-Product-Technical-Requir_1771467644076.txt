# 7Ô∏è‚É£ ADAPTIVE LEARNING ALGORITHM

Use simplified SM-2 Spaced Repetition.

---

## When Answer is Correct

```
if correct:
    correct_streak += 1
    ease_factor += 0.1 (if confidence high)
    interval = interval * ease_factor
```

Update:

```
next_review = now + interval days
```

---

## When Answer is Wrong

```
correct_streak = 0
wrong_count += 1
interval = 1
next_review = now + 1 day
```

---

# 8Ô∏è‚É£ WORD SELECTION ALGORITHM

When generating quiz:

Score each word:

```
score =
    difficulty_weight
  + days_since_last_seen
  + (wrong_count * 2)
  - correct_streak
```

Sort descending.
Pick top N.

This ensures:

* Weak words appear more
* New words appear
* Mastered words disappear

---

# 9Ô∏è‚É£ MASTERING LOGIC

| Correct Streak | Mastery Level |
| -------------- | ------------- |
| 1              | Learning      |
| 3              | Familiar      |
| 5              | Strong        |
| 7              | Mastered      |

If mastered:

* Show only during review cycles
* Not in regular quiz pool

---

# üîü QUIZ OPTION GENERATION LOGIC

For a word:

1 correct answer
3 distractors

Distractor priority:

1. Same cluster words
2. Same part of speech
3. Similar transliteration
4. Random fallback

This increases difficulty intelligently.
